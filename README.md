## Get Started

1. Install dependencies: `yarn`.
2. Run `yarn dev` and visit <https://localhost:5173>.

## Linting

- `yarn lint` - checks for eslint, prettier, and TS errors

CI/CD pipeline automatically runs linter checks on push.
